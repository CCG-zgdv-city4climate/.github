# City4Climate — software ecosystem maintained by CCG/ZGDV

This GitHub organization aggregates the repositories made available for the City4Climate initiative, with a focus on controlled sharing of source code, technical artefacts, and supporting documentation relevant to project partners and promoters.

The development backbone remains maintained internally, while this GitHub space is used to organize and expose the repository landscape associated with the solution delivered by **Instituto CCG/ZGDV**.

## About City4Climate

City4Climate is a digital twin initiative oriented to climate action, combining urban data integration, processing, simulation, analytics, and user-facing services.

Within this context, **CCG/ZGDV** is responsible for the design and development of core software components, integration artefacts, and supporting technical assets that sustain the operational prototype and its evolution.

## Purpose of this organization

This organization is intended to:

- provide a structured view of the City4Climate software ecosystem;
- make selected repositories available to authorized stakeholders;
- preserve the technical history of the delivered components;
- support navigation across the repositories that compose the solution.

## Repository landscape

The repositories in this organization represent different layers of the City4Climate software ecosystem, including:

- user-facing components;
- analytical and simulation services;
- adapters and processors for data ingestion and transformation;
- contextual and historical data components;
- gateway and integration services;
- management and transversal documentation assets.

## Suggested entry points

The following repositories are the main starting points for understanding the solution:

| Repository | Role |

## Working model

This GitHub organization is used as a controlled dissemination and collaboration space for the repositories selected for sharing in the context of City4Climate.

Only the repositories, branches, and artefacts considered appropriate for this purpose are made available here.

## Ownership and attribution

The repositories hosted in this organization reflect software work developed by **Instituto CCG/ZGDV** within the City4Climate project, delivered to ARTE as the adopting public entity in the scope of City4Climate – Gémeos Digitais Urbanos para a Ação Climática, funded by Agência para a Reforma Tecnológica do Estado (ARTE) under tender 2025.00141.DT4ST, Digital Twins4SmartTerritories (DT4ST): Gémeos Digitais para Territórios Inteligentes, supported by the Portuguese Republic’s Recovery and Resilience Plan (PRR), through investment component TD-C19-i08, “Territórios Inteligentes”.
